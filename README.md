# Chatting Style Message Box 


## Explain

I use react(cra), typescript, scss and mui(material-ui).

1. npx create-react-app csFront –template typescript
2. yarn add sass
3. yarn add @mui/material @emotion/react @emotion/styled @mui/icons-material
